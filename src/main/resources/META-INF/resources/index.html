    • You can find the code on GitHub

    This application is using the smallrye health quarkus extensions for liveness and readyness probes.

    The docker images can be found on quay.io

    Services

    The following endpoints are available:

    • /q/health/live Lifeness check via smallrye
    • /q/health/ready Readyness check via smallrye
    • /q/metrics Metrics via smallrye
    • /hello The app
    • /faker The fake name generator app
